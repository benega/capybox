# Capybox – Project Overview

## 🎯 Vision
Capybox is a social gaming platform designed for families and groups to play together instantly using their TVs and mobile devices.

The goal is to create a frictionless, fun, and accessible gaming experience where players can join sessions via QR code and use their phones as controllers—no installation required.

Inspired by the social harmony of capybaras, Capybox represents a space where everyone can play together.

---

## 🧩 Core Concept

- TV = Host (Game Screen)
- Mobile Devices = Controllers
- QR Code = Instant Join
- PWA-first = Zero friction

Players scan a QR code displayed on the TV and instantly join the game session using their phones.

---

## 💡 Value Proposition

- No downloads required for players (PWA)
- Instant multiplayer via QR code
- Designed for families, friends, and casual gatherings
- Simple, fun, and social gameplay
- Scalable platform for multiple game types (not only quizzes)

---

## 🎮 Gameplay Experience

1. Host opens Capybox on a TV browser
2. A QR code is displayed
3. Players scan the QR code using their phones
4. Players join the session instantly
5. Host selects a game
6. Players interact using their phones as controllers

---

## 🧠 Product Strategy

Capybox is not a single game—it is a platform for social games.

### Initial Focus (MVP)
- Simple multiplayer games (e.g., quiz-style)
- Fast onboarding and session creation
- Real-time interactions

### Future Expansion
- Party games
- Competitive and cooperative modes
- Custom game creation
- Content marketplace
- Subscription model (no ads + premium games)
- Optional game purchases
